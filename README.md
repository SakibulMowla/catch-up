# catch-up

## Prerequisite

- Node.js
- npm

## Setup

- Clone the repo
- `npm install`
- Setup the aws credentials as environment variables

## Running test

- Run all tests: `npm test`
- Run a particular test: `npx jest <testname>`

## Lint

- check lint error: `npm run lint`
- autofix lint error: `npm run lint-fix`
